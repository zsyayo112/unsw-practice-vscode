// WebView → 插件（用户操作）
type WebviewMessage =
  | { type: 'ready' }                                    // WebView 加载完成
  | { type: 'submit'; code: string; problemId: string }  // 提交代码
  | { type: 'run'; code: string; problemId: string }     // 仅运行（不计入提交）
  | { type: 'saveCode'; code: string; slug: string }     // 自动保存

// 插件 → WebView（数据推送）
type ExtensionMessage =
  | { type: 'problem'; data: Problem }           // 传题目数据
  | { type: 'result'; data: SubmissionResult }   // 传提交结果
  | { type: 'loading'; value: boolean }          // 加载状态
  | { type: 'authRequired' }                     // 需要登录
```

## 七、关键技术决策 & 原因

| 决策 | 选择 | 原因 |
|------|------|------|
| 打包工具 | esbuild | 比 webpack 快 10 倍，配置简单 |
| 代码编辑器 | Monaco CDN | 不打包进插件，节省体积 |
| 密钥存储 | SecretStorage | VSCode 官方推荐，加密存储 |
| 代码执行 | Piston API | 免费，支持 Python，无需自建沙箱 |
| 样式方案 | CSS Variables | 自动适配 VSCode 亮色/暗色主题 |
| Mock 模式 | MOCK_MODE flag | API 未完成时可独立开发 UI |

## 八、MVP 边界（先做什么，不做什么）

**MVP 包含：**
- 题目列表展示（Mock 数据）
- 做题面板（Monaco 编辑器 + 题目描述）
- 代码运行和结果展示
- 本地进度保存（localStorage）

**MVP 不包含（留给后续版本）：**
- 用户登录和云端同步
- 排行榜
- 讨论区
- 多语言支持（先只支持 Python）
```
